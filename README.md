# Employee Engagement, Satisfaction, and Burnout Diagnostic Analysis

## Project Overview
This project analyzes employee engagement, satisfaction, burnout risk and work-life balance patterns at Palo Alto Networks using Power BI.

## Problem Statement
Organizations often detect attrition too late. This project identifies burnout and engagement deterioration early using HR analytics.

## Technologies Used
- Power BI
- Power Query
- DAX
- Excel

## Dashboard Modules
### 1. Engagement Health Overview
- Organization engagement KPI
- Satisfaction distribution
- Department analysis

### 2. Burnout Risk Dashboard
- Burnout analysis
- Overtime impact
- Travel stress analysis

### 3. Role & Career Stage Analysis
- Job-level engagement
- Career stage trends

### 4. Manager Action Panel
- Low engagement alerts
- Priority intervention areas

## KPIs
- Engagement Index
- Burnout Risk Score
- Work-Life Balance Index
- Satisfaction Stability Score
- Workload Stress Indicator
